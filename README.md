# Онлайн-дневник тренировок

## Описание проекта
Онлайн-дневник тренировок — это веб-приложение для учета физических тренировок пользователя.  
Приложение позволяет добавлять, просматривать, редактировать и удалять тренировки, а также анализировать прогресс и статистику.  
Реализована система ролей: пользователь, тренер и администратор.

---

## Основные функции программы

### Пользователь
- Регистрация и вход в систему
- Добавление тренировок (дата, упражнение, подходы, повторения, вес)
- Редактирование и удаление своих тренировок
- Просмотр общего объёма тренировок
- Просмотр прогресса по упражнениям
- Просмотр статистики по месяцам
- Просмотр комментариев тренера

### Тренер
- Просмотр тренировок пользователей
- Добавление комментариев к тренировкам

### Администратор
- Просмотр списка пользователей
- Удаление пользователей и их тренировок

---

## Архитектура проекта (MVC)

- **Model**  
  Классы `User`, `Workout`, `Comment` — описывают структуру базы данных и бизнес-логику

- **View**  
  HTML-шаблоны в папке `templates/` (Jinja2)

- **Controller**  
  Маршруты Flask в файле `app.py`, обрабатывающие запросы пользователя
